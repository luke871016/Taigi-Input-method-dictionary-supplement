建中的[教育部臺灣台語輸入法](https://language.moe.gov.tw/result.aspx?classify_sn=23&subclassify_sn=442)詞庫增補檔案

目前包含下跤幾个表，逐家通看家己的需求匯入家己的輸入法詞庫

## 匯入方式

> [!CAUTION] 
> 注意：匯入進前建議先備份家己的語詞管理資料，避免匯入效果無滿意想欲取消

1. 揤「語詞管理」
2. 揤「匯入」
3. 揀你欲匯入的資料表
4. 完成

## 資料表說明

### 一府五院
總統府、行政院、立法院、司法院、監察院、考試院的組織相關機關名稱

### 數字、時間、日期
遵照台羅手冊的所有連寫的數字詞、數量詞
閣有時間、日期相關的詞

### 台／臺
目前輸入法詞庫有收錄的「臺」相關詞的「台」版本

### 內政部菜市仔名
內政部名姓統計資料的頭前100名菜市仔名的台語讀音

### 教典僻智識
建中寫教典僻智識捷用的一寡語詞
